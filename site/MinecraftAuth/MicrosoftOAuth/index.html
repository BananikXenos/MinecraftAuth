    MicrosoftOAuth

    class MicrosoftOAuth

    MicrosoftOAuth class for handling Microsoft OAuth flows.

    This class provides functionalities for Microsoft OAuth, including device authentication, token retrieval, XBL authentication, XSTS authentication, and Minecraft authentication.

    Functions

    Name Description
    DeviceAuth Initiates device authentication flow.
    StartDeviceAuth Starts the device authentication process.
    CheckDeviceAuth Checks the status of device authentication.
    RefreshMicrosoftAuth Refreshes the Microsoft authentication using the provided refresh token.
    LoginToAccount Logs in to a Minecraft account using Microsoft authentication details.
    ValidateAccount Validates the Minecraft account and refreshes the token if necessary.
    RefreshMinecraftToken Refreshes the Minecraft access token for the given account.
    RefreshMicrosoftToken Refreshes the Microsoft access token for the given account.
    AuthenticateWithXBL Authenticates with Xbox Live (XBL) using the provided authentication token.
    AuthenticateWithXSTS Authenticates with Xbox Secure Token Service (XSTS) using XBL authentication details.
    AuthenticateWithMinecraft Authenticates with Minecraft services using XBL and XSTS authentication details.
    GetMinecraftProfile Retrieves the Minecraft profile using the provided Minecraft authentication details.
